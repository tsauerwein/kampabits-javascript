022-while.htm

for and while

All loops can be written as for or as while loop.

 

  • Use for loops, if you know how often you'll loop.
  • Use while loops, if you don't know how often you'll loop.

Game: Make-a-guess


var numberToGuess = 35;

var numberFound = false;
while(!numberFound) {
	var number = prompt("Make a guess!");

	if (number < numberToGuess) {
		alert("You entered " + number + ". " 
			+ "The number we are looking for is bigger, try again!");
	} else if (number > numberToGuess) {
		alert("You entered " + number + ". " 
			+ "The number we are looking for is smaller, try again!");
	} else {
		numberFound = true;
		alert("Yes!");
	}
}
